<h3>Instalação</h3>

Primeiro utilize o comando abaixo:

> $ git clone https://github.com/mauriani/clone-netflix-web

Depois disso acesse a pasta de cada projeto: 

> $ yarn install # ou npm install

Para executar os projetos:

> $ yarn start
